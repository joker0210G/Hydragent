Hydragent

Privacy-first AI agent runtime. Runs on your machine, your data stays on your machine. No telemetry, no cloud account required.

Install in one command

Windows (PowerShell 5.1+ / 7+) 
irm https://hydragent.dev/install.ps1 | iex

Or: open PowerShell, paste the line above, press Enter. That's it.

macOS / Linux (bash / zsh) 
curl -fsSL https://hydragent.dev/install.sh | sh

The installer downloads a prebuilt binary, drops a hydragent command on your PATH, and launches the first-run onboarding wizard. Re-run the same command any time to update.

What you get

  • hydragent serve — start the agent runtime
  • hydragent chat — interactive REPL
  • hydragent status / hydragent ps — inspect state
  • hydragent stop [pid] — shut down gracefully
  • hydragent onboard — re-run the first-run wizard

Privacy by default

  • All state, memory, and logs are stored locally in ~/.hydragent/.
  • No outbound telemetry, no phone-home, no auto-update beacon.
  • You choose which model provider to talk to — local Ollama, OpenRouter, OpenAI, Anthropic, or a custom endpoint.
